# Week1_WinForms_BasicApp

## 概要
C#の基礎学習（Week1）として作成した Windows Forms アプリケーションです。  
Consoleアプリで作成した処理ロジックを、WinForms（GUI）から呼び出す形に作り替えています。

入力された文字列をもとに処理を行い、結果を画面に表示します。

---

## 開発環境
- OS：Windows
- 言語：C#
- フレームワーク：.NET 6（または .NET 7）
- UI：Windows Forms
- IDE：Visual Studio

---

## 機能一覧
- 文字列入力
- 入力チェック（未入力・空白の場合はエラー表示）
- 入力内容の表示
- 入力文字数の計算・表示

---

## 画面イメージ
- テキストボックスに文字を入力
- 「実行」ボタンをクリック
- 処理結果を画面下部に表示

---

## 処理内容
1. ユーザー入力を取得
2. 入力値を処理クラスに渡す
3. 入力チェックを実施
4. 入力文字数を計算
5. 結果を文字列として返却
6. 画面に表示

---

## 設計のポイント
- UI（Form）とロジック処理を分離
- 処理は static クラスに切り出し、再利用可能な構成
- UI層ではロジックを持たず、表示制御のみを担当
